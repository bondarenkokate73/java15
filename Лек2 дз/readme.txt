Для запуска приложения нажать на ярлык JavaApplication1.jar или открыть программу в папке JavaApplication1/dist/JavaApplication1.jar
При успешном создании программой странички html, она автоматически откроется после запуска программы.

**************************************************************************************************************************************

При желании, можно изменить данные для вывода на html-страничку: в папке JavaApplication1/dist/ открыть файл info.properties c помощью блокнота, поменять данные, не нарушая структуры файла {ключ}={значение}, затем сохраняем файл и запускаем программу.
